# Bomberman

## Requirements

### _Environments:_

- **Docker**  

  or

- **Linux/MacOS**
- **gcc**
- **make**
- **SDL2**
- **SDL2 image and ttf**

### _Build and run:_

- **$> git clone {projet}**
- **$> cd {projet}**

Or if using docker:
- **$> docker build -t bomberman .**
- **$> docker run --rm -it -v ${PWD}:/app sdl_project**

Next:
- **$> cd {projet}**
- **$> make**
- **$> ./bomberman**
